# T. Morris Painting & Decorating Website

A custom-built responsive website for a professional painter and decorator operating in Oxfordshire. This site showcases completed projects, client testimonials, and provides contact details.

---

## 📌 Features

- Responsive design for mobile, tablet, and desktop
- Portfolio gallery with smooth scroll and slideshow
- Client testimonial section styled as review cards
- Contact page with accessible form and click-to-call support
- SEO-friendly meta tags for visibility
- Lightweight, semantic HTML and CSS

---

## 🚀 Getting Started

1. Clone this repository:

```bash
git clone https://github.com/your-username/painting-decorating-site.git
```

2. Open `index.html` in your browser to preview locally.

---

## 🧱 Tech Stack

- HTML5
- CSS3
- JavaScript (vanilla)
- Responsive layout using Flexbox and CSS Grid
- WebP images for performance
