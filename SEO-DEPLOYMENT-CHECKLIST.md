# SEO Deployment Checklist

## Before Publish

1. Confirm all updated files are uploaded:
- index.html
- contact.html
- portfolio.html
- testimonial.html
- 404.html
- sitemap.xml
- robots.txt

2. Verify canonical URLs in each page source:
- Home uses `https://tmorrispainting.co.uk/`
- Internal pages use full absolute canonical URLs

3. Verify metadata is unique per page:
- title
- description
- og:title
- og:description

4. Verify schema scripts are present and valid:
- LocalBusiness on core pages
- FAQPage on homepage
- Review graph on testimonial page

## After Publish

1. Open and verify these URLs load correctly:
- https://tmorrispainting.co.uk/
- https://tmorrispainting.co.uk/portfolio.html
- https://tmorrispainting.co.uk/testimonial.html
- https://tmorrispainting.co.uk/contact.html
- https://tmorrispainting.co.uk/404.html
- https://tmorrispainting.co.uk/sitemap.xml
- https://tmorrispainting.co.uk/robots.txt

2. Search Console setup:
- Add and verify site property
- Submit sitemap: `https://tmorrispainting.co.uk/sitemap.xml`
- Request indexing for homepage, portfolio, testimonial, contact

3. Validate structured data:
- Run each page through Rich Results Test
- Check FAQPage on homepage
- Check Review and LocalBusiness on testimonial page

4. Validate social cards:
- Test homepage and contact page in Open Graph/Twitter validators

5. Basic performance sanity check:
- Run PageSpeed Insights on homepage and portfolio
- Confirm no large layout shifts in portfolio image grid

## Ongoing Monthly SEO Tasks

1. Add new project photos to portfolio with descriptive alt text.
2. Add one new client testimonial when available.
3. Keep sitemap updated when adding or removing pages.
4. Re-check Search Console for indexing or enhancement warnings.
5. Refresh homepage FAQ with real customer questions.
